左上（输入信息）：
股票编码/名称-1
手数/价格-2
涨停板/大于-3
跌停板/小于-4

左下（自选列表）：
条件：1不为空
功能：增加（快捷键：Enter）、删除
描述：每1s刷新最新价格信息

右上（封单检测）：
条件：1、2、3/4不为空
功能：增加（快捷键：Shift+Enter） 删除全部
描述：每1s检测当前股票封单是否小于手数，小于则弹窗提醒。

右下（价格策略）：
条件：1、2、3/4不为空
功能：增加（快捷键：Ctrl+Enter） 删除
描述：每1s检测当前股票是否大于/小于设置价格，到达则弹窗提醒。
